# podcast-studio-av-system-design
Audio, video, and network architecture for a multi-camera podcast studio

## System Overview

The studio supports:

- Multi-camera video production
- Live streaming
- Dante audio networking
- NDI video distribution
- PTZ camera control
- Lighting control via DMX

## Key Equipment

- Blackmagic ATEM 4 M/E Constellation 4K
- Three Canon XF605 UHD Camcorders
- 2 BirdDog NDI cameras
- Dante audio network
- Tesira Forte DSP
- vMix streaming workstation

## Tools Used

- diagrams.net (draw.io)
- Blackmagic ATEM
- Dante Controller
- Biamp Tesira Software
- NDI
- vMix

## Purpose

This project demonstrates broadcast system design and AV infrastructure architecture.
